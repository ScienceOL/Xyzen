# 技能

技能是可复用的指令包，赋予智能体专门的能力。它们使用 SKILL.md 格式编写，可以附加到任何智能体上。 {{ className: 'lead' }}

## 什么是技能？

技能是一个结构化的 Markdown 文档（SKILL.md），定义了：

- 智能体应该做什么的指令集
- 何时触发这些指令的上下文
- 可选的示例和约束条件

当技能附加到智能体时，其指令会自动注入到智能体的系统提示词中，赋予智能体新的能力而无需修改其核心配置。

## SKILL.md 格式

一个技能文件遵循以下结构：

```markdown {{ title: 'SKILL.md' }}
---
name: 翻译助手
description: 将文本翻译为多种语言
version: 1.0.0
tags: [翻译, 语言]
---

# 翻译助手

你是一个翻译专家。当用户要求翻译时：

1. 识别源语言
2. 翻译为目标语言
3. 提供翻译注释和备选方案

## 约束

- 保持原文的语气和风格
- 对于专业术语，提供原文注解
```

YAML 前置信息（`---` 之间的内容）包含元数据。主体部分是注入到系统提示词中的实际指令。

## 创建技能

1. 在 Xyzen 界面中导航至 **技能** 页面
2. 点击 **创建技能**
3. 编写你的 SKILL.md 内容
4. 保存并为技能添加名称和描述

你也可以从 SKILL.md 文件导入技能。

## 将技能附加到智能体

1. 打开智能体设置
2. 找到 **技能** 部分
3. 从可用技能列表中选择技能
4. 保存更改

附加多个技能时，它们的指令会按顺序合并到系统提示词中。

## 技能市场

你可以在[市场](/marketplace)中发布你的技能，让其他用户发现和使用。

- **发布** — 将你的技能分享到市场
- **安装** — 从市场安装其他用户的技能
- **复刻** — 复制并自定义他人的技能
