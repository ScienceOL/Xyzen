# 知识库

知识库允许智能体在对话过程中访问你的文档和数据，通过检索增强生成（RAG）实现基于你的数据的准确回答。 {{ className: 'lead' }}

## 概述

知识库是一个文档集合，经过处理并编入索引后供智能体检索。当智能体需要回答关于你的数据的问题时，它会搜索关联的知识库，找到相关段落，并将其用作回答的上下文。

## 创建知识库

1. 导航至 **知识库** 页面
2. 点击 **创建知识库**
3. 输入名称和描述
4. 上传你的文档

## 上传文件

支持的文件格式：

- **文档** — PDF、Word (.docx)、PowerPoint (.pptx)、文本文件
- **代码** — 任何纯文本代码文件
- **数据** — CSV、JSON

上传后，文件会自动处理：

1. **解析** — 提取文本内容
2. **分块** — 将文本拆分为语义片段
3. **嵌入** — 生成向量嵌入
4. **索引** — 存储到向量数据库中

## 关联到智能体

在智能体设置中选择知识库进行关联：

1. 打开智能体设置
2. 找到 **知识库** 部分
3. 选择要关联的知识库
4. 保存更改

一个智能体可以关联多个知识库。智能体会在需要时自动搜索所有关联的知识库。

## 检索原理

当智能体使用 `knowledge_search` 工具时：

1. 用户问题被转换为向量嵌入
2. 在知识库中搜索最相似的文档片段
3. 返回最相关的片段和来源信息
4. 智能体使用这些片段作为上下文生成回答

检索结果包含来源文件和页码引用，智能体会在回复中引用这些来源以保证信息的可追溯性。
