# 定时任务

定时任务让智能体能够按照预设的时间表自动运行，实现每日简报、定期检查等自动化场景。 {{ className: 'lead' }}

## 概述

定时任务将智能体与时间表绑定。到达预定时间时，系统自动向指定的智能体发送消息并触发执行。结果会保存在对话历史中。

## 计划类型

Xyzen 支持两种计划类型：

### Cron 表达式

使用标准的 cron 表达式精确控制执行时间：

```
# 每天早上 9 点
0 9 * * *

# 每周一下午 2 点
0 14 * * 1

# 每 6 小时
0 */6 * * *
```

### 自然语言

你也可以用自然语言描述计划（由 LLM 解析为 cron 表达式）：

- "每天早上 9 点"
- "每周五下午 5 点"
- "每 3 小时执行一次"

## 创建任务

1. 导航至 **定时任务** 页面
2. 点击 **创建任务**
3. 选择要执行的智能体
4. 设置时间表（cron 或自然语言）
5. 输入要发送给智能体的消息
6. 保存

## 智能体创建的任务

智能体也可以通过 `schedule_task` 工具自己创建定时任务。例如，你可以告诉智能体：

> "每天早上帮我总结昨天的新闻"

智能体会自动创建一个定时任务来执行这个请求。

## 管理任务

在定时任务页面你可以：

- **查看** — 查看所有定时任务及其执行历史
- **暂停 / 恢复** — 临时停止或恢复任务
- **编辑** — 修改时间表或消息内容
- **删除** — 永久删除任务
