# 智能体

智能体是 Xyzen 的核心构建模块。每个智能体都是一个由 LLM 驱动的工作流，能够使用工具、访问知识库并与其他智能体协作。 {{ className: 'lead' }}

## 智能体类型

### ReAct 智能体

ReAct（推理 + 行动）是默认的智能体类型。它采用工具调用循环运行：

1. LLM 接收用户消息和可用工具列表
2. LLM 决定调用哪些工具（或直接回复）
3. 工具执行后结果返回给 LLM
4. LLM 基于工具结果进行推理，然后调用更多工具或生成最终回复

ReAct 智能体适用于大多数通用任务：回答问题、搜索网络、编写代码和分析文档。

### Deep Research 智能体

Deep Research 智能体专为需要深入研究的复杂问题设计。它由四个阶段组成：

1. **澄清** — 分析问题，向用户确认范围
2. **简报** — 制定研究计划，明确搜索查询和来源
3. **监督** — 协调多轮搜索和分析
4. **报告** — 综合研究结果，生成结构化报告

### 自定义智能体

你可以使用图构建器创建自定义智能体类型。自定义智能体由可复用组件组成，支持任意的图工作流。参见 [自定义智能体](/custom-agents) 了解详情。

## 创建智能体

在 Xyzen 界面中创建新智能体：

1. 点击侧边栏中的 **+** 按钮
2. 输入名称和描述
3. 选择智能体类型（ReAct、Deep Research 或自定义）
4. 选择 LLM 模型（例如 GPT-4o、Claude Sonnet）
5. 启用你需要的工具（网络搜索、代码执行等）

你还可以配置系统提示词来自定义智能体的行为和性格。

## 智能体配置

| 字段 | 说明 |
|------|------|
| `name` | 显示名称 |
| `description` | 功能描述，用于智能体选择 |
| `agent_type` | `react`、`deep_research` 或 `custom` |
| `model` | LLM 模型标识符 |
| `system_prompt` | 自定义系统指令 |
| `tools` | 启用的工具 ID 列表 |
| `knowledge_sets` | 关联的知识库 ID |
| `skills` | 附加的技能 ID |
| `temperature` | LLM 温度参数（0-2） |
| `max_tokens` | 最大输出 token 数 |

## 内置智能体

Xyzen 预置了以下系统智能体：

- **CEO / Root 智能体** — 内置的通用 ReAct 智能体，预配置了网络搜索、代码执行和知识管理等常用工具。推荐新用户使用
- **Deep Research** — 系统级的 Deep Research 智能体，用于深入的研究任务
- **Slides** — 专门用于创建演示文稿的智能体

## 智能体作用域

智能体有两种作用域：

- **系统智能体** — 由管理员创建，所有用户可见。包括内置智能体和管理员发布的智能体
- **用户智能体** — 由用户创建，仅创建者可见。用户可以从市场复刻系统智能体并自定义
