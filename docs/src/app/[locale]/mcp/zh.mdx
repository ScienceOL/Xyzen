# MCP 服务

Model Context Protocol（MCP）允许 Xyzen 连接外部工具提供商，无需修改代码即可扩展智能体功能。 {{ className: 'lead' }}

## 什么是 MCP？

MCP（模型上下文协议）是一个开放标准，用于将 AI 模型连接到外部工具和数据源。MCP 服务器暴露一组工具，Xyzen 通过标准协议连接并使用这些工具。

这意味着你可以将任何支持 MCP 的工具提供商连接到 Xyzen，而无需编写自定义集成代码。

## 添加 MCP 服务器

1. 导航至 **MCP 服务器** 页面
2. 点击 **添加服务器**
3. 输入服务器 URL 和配置
4. 测试连接
5. 保存

Xyzen 支持 SSE 和 Streamable HTTP 两种传输方式。

## Smithery 注册中心

[Smithery](https://smithery.ai) 是一个 MCP 服务器注册中心。Xyzen 内置了 Smithery 集成，让你可以直接浏览和安装社区发布的 MCP 服务器。

要使用 Smithery：

1. 在环境变量中配置 Smithery API Key（`XYZEN_MCP_Smithery_Key`）
2. 在 MCP 页面浏览可用的服务器
3. 一键安装

## 测试工具

添加 MCP 服务器后，你可以在 MCP 管理页面测试每个工具：

1. 选择要测试的工具
2. 填入测试参数
3. 点击 **运行** 查看结果

## 在智能体中使用 MCP 工具

MCP 工具添加后会自动出现在工具列表中。在智能体设置中启用它们即可：

1. 打开智能体设置
2. 在工具列表中找到 MCP 工具（带有服务器前缀）
3. 启用所需的工具
4. 保存更改

智能体在对话中会像使用内置工具一样使用 MCP 工具。
