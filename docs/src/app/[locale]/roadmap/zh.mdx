# 路线图

Xyzen 的发展历程和未来方向概览。 {{ className: 'lead' }}

## 近期发布

### 多智能体编排
- ReAct 和 Deep Research 智能体类型
- 子智能体委派与上下文传递
- 智能体市场，支持发布和复刻
- 可复用的组件系统用于图构建

### 实时流式传输
- 基于 SSE 的流式传输（替代 WebSocket）
- 使用 Redis Streams 进行事件持久化和断线重连
- 以 60fps 的帧率进行数据块缓冲，实现流畅的 UI 更新

### 知识与记忆
- 知识库管理，支持文件上传
- 核心记忆系统（用户摘要、偏好、活跃上下文、工作规则）
- 从用户修正中持续学习

### 沙箱与代码执行
- 隔离的代码执行环境
- 文件操作和 IDE 预览
- Daytona 后端集成

### 平台功能
- 通过 Casdoor 进行 OAuth 认证
- MCP 服务器集成与 Smithery 注册
- 使用 SKILL.md 格式的技能系统
- 定时任务自动化
- 订阅与支付系统
- Web 推送通知

## 进行中

- **文档站点** — 面向用户和开发者的全面文档（你正在阅读！）
- **SSE 迁移** — 完成所有实时功能从 WebSocket 到 SSE 的迁移
- **智能体性能** — 优化图编译和执行延迟

## 计划中

- **智能体间通信** — 多智能体工作流中智能体之间的直接消息传递
- **工作流模板** — 针对常见用例的预构建多步工作流
- **插件 SDK** — 简化的 API，用于构建自定义工具和集成
- **增强 RAG** — 改进的检索增强生成，支持混合搜索
- **协作** — 多用户共享会话和工作空间
- **可观测性** — 增强的智能体执行追踪和调试
- **移动应用** — iOS 和 Android 原生移动体验
