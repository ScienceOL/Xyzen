# 分享

通过公开链接分享有趣的智能体对话，让他人查看你的 AI 交互内容。 {{ className: 'lead' }}

## 分享对话

要分享一段对话：

1. 打开你想分享的对话
2. 点击对话标题栏中的 **分享** 按钮
3. 确认分享
4. 复制生成的公开链接

任何拥有该链接的人都可以查看这段对话，无需登录。

## 分享内容

分享的对话包含：

- 用户消息和智能体回复的完整内容
- 智能体使用的工具调用和结果
- 代码块、图表和其他富文本内容
- 智能体的思考过程（如果有）

不会分享的内容：

- 你的账户信息和个人资料
- 智能体的系统提示词和配置
- 知识库的原始文档
- 其他对话的内容

## 管理分享

在 **分享** 页面管理你已分享的对话：

- **查看** — 查看所有已分享的对话
- **取消分享** — 撤销某段对话的公开访问，链接将失效
- **复制链接** — 重新复制分享链接
