# 市场

市场是用户分享智能体和技能的平台。在这里你可以发现、安装和复刻其他用户创建的智能体和技能。 {{ className: 'lead' }}

## 浏览

在侧边栏中打开 **市场** 页面，浏览社区发布的智能体和技能。你可以：

- 按类别筛选（通用、研究、编程、创作等）
- 按热度或最新排序
- 搜索特定的智能体或技能

每个条目显示名称、描述、作者和安装量。

## 发布智能体

将你创建的智能体分享到市场：

1. 打开智能体设置
2. 点击 **发布到市场**
3. 填写描述和标签
4. 确认发布

发布后，其他用户可以发现并复刻你的智能体。你的智能体配置（系统提示词、工具列表、技能等）会被复制，但对话数据不会分享。

## 复刻智能体

从市场安装智能体：

1. 在市场中找到你感兴趣的智能体
2. 点击 **复刻**
3. 智能体会被复制到你的账户中

复刻后你可以自由修改智能体的配置。原作者的后续更新不会自动同步到你的副本。

## 技能市场

技能也可以在市场中分享：

- **发布技能** — 将你的 SKILL.md 发布到市场
- **安装技能** — 浏览并安装社区技能
- **复刻技能** — 复制并自定义他人的技能

技能和智能体共享同一个市场平台，但在不同的标签页中展示。
